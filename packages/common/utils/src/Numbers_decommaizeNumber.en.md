# decommaizeNumber

Removes commas from [commaized](https://slash.page/libraries/common/utils/src/Numbers_commaizeNumber.i18n) number.

```typescript
function decommaizeNumber(numStr: string): number;
```

## Example

```typescript
decommaizeNumber('13,209,802'); // 13209802
```
