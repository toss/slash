# parseName

한글 이름을 `[성, 이름]`으로 분류하는 parser 입니다.

- 다음과 같은 규칙을 따릅니다:
  - 두 글자 이름: `[성, 이름]`
  - 세 글자 이름:
    - 두 글자 성씨: `['', 풀네임]`
    - 한 글자 성씨: `[성, 이름]`
  - 그 외: `['', 풀네임]`

## Example

```typescript
parseName('김토스'); // => ['김', '토스']
parseName('이도'); // => ['이', '도']
parseName('김나박이'); // => ['', '김나박이']
parseName('John Doe'); // => ['', 'John Doe']
```
