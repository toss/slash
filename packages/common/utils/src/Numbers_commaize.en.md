# commaizeNumber

Separates the given value by comma.

```typescript
function commaizeNumber(value: number | string): strubg;
```

## Example

```typescript
commaizeNumber(13209802); // '13,209,802'
commaizeNumber('13209802'); // '13,209,802'
```
